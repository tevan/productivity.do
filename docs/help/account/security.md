# Email and password

## Changing your email

Settings → Account → Profile → Email → "Change"

We send a confirmation link to the new email. The change takes effect
when you click it. The old address gets a notification.

If you no longer have access to the old email, contact
support@productivity.do — we'll verify your identity another way.

## Changing your password

Settings → Account → Profile → Password → "Change"

Requires the current password. New password must be ≥12 characters.

## Forgot your password?

[Reset link](/login) → "Forgot password" → enter email. We send a reset
link valid for 1 hour.

## Active sessions

Settings → Account → Profile → Active sessions

Lists every device/browser currently signed in. Each entry shows:
- Device fingerprint (OS + browser)
- IP address (last seen)
- Approximate location
- Last activity

You can sign out individual sessions or "Sign out everywhere" to nuke
all sessions except the current one.

## Two-factor authentication

Coming soon. We're prioritizing it after the public launch.

## What happens if I'm locked out

- Forgot password → reset via email
- Email no longer works → contact support
- 2FA enabled and lost device → contact support; we'll verify identity

We never share account credentials with anyone. We will never email you
asking for your password.
