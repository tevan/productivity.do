# Notes

Long-form text notes for things that don't fit as a calendar event or
task. Each note has a title, body (markdown), optional pinned flag,
and a color.

## Creating a note

- `N` in Notes view
- Type in the sidebar's note quick-add

## Editing

Click any note in the sidebar to open it in the reader pane. Click the
title or body to edit inline. Save is automatic.

## Markdown supported

- Headings (`#` `##` `###`)
- Bold, italic, links
- Lists (ordered + unordered)
- Code blocks (\`\`\`)
- Blockquotes (`>`)

## Search

Cmd+F searches across event/task/note content. Notes are full-text
indexed.

## Pinning

Click the pin icon on a note to keep it at the top of the list.

## Color tags

Click the color dot in the reader pane to assign one of 12 colors.
Useful for grouping related notes by theme.
