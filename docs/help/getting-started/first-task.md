# Your first task

Two ways:

1. **Press `N`** in the Tasks view — full editor
2. **Type in the quick-add input** at the top of the sidebar — Todoist-style natural language: `"Submit timesheet tomorrow @errands p1"` (date, label, priority all parsed inline)

## Task properties

- **Title** — required; supports markdown formatting
- **Due date / due time** — optional. "Today", "Tomorrow", "Next Mon" all work
- **Priority** — 1 (urgent) to 4 (low). Affects sort order
- **Project** — group tasks. Native projects work without Todoist
- **Labels** — multiple per task; useful for filtering
- **Description** — markdown supported

## Two views — list and board

- **List** — group by date, project, label, or priority. Multi-select with shift-click. Bulk actions: Complete / Push to today / Push to tomorrow / Delete
- **Board** — kanban with drag-to-move-between-columns. Default columns: To Do, In Progress, Done. Up to 5 custom columns

Toggle between them in the Tasks view header.

## Auto-scheduling

Click the calendar icon on any task to "auto-schedule" — finds the next
free slot in your work hours and creates a calendar event for it.

## Drag tasks to the calendar

In Calendar view, drag a sidebar task onto a day to set its due date,
or onto a time slot to set a specific due time.

## Subtasks

Tasks can have a parent. Indented under their parent in the list.

## Recurring tasks

Set "Repeats" in the editor. Once completed, the task auto-resets to
the next occurrence.
