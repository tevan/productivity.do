# Your first event

Three ways to create an event:

1. **Click an empty time slot** in Day or Week view — opens a quick popover
2. **Press `N`** anywhere — opens the full editor
3. **Click "+ New" in the toolbar** — same as `N`

Required fields: title, start, end. Everything else (location,
description, attendees, color) is optional.

## Editing an event

- **Single-click** — small popover (change time, add notes)
- **Double-click** — full editor modal
- **Drag the body** — move to a different time
- **Drag the bottom edge** — resize duration
- **Right-click** — context menu (Duplicate, Move to calendar, Delete, etc.)

## Recurring events

Editor has a "Repeats" dropdown: Daily, Weekly, Monthly, Yearly, plus
custom RRULE syntax. When you **edit or delete** a recurring event, you'll
be asked whether to apply the change to:
- This event only (default — safest)
- This and all following events
- All events in the series

The same prompt appears for keyboard delete and right-click → Delete.

## Multi-calendar

If you have several calendars connected (work, personal, family), pick
which one in the editor's Calendar dropdown. The event lands there and
syncs back to that source.

## Tips

- Press `T` to jump to today, anywhere
- Press `G` to open a date picker
- Press `F` to find a free time across multiple calendars
- Press `1`-`4` to switch calendar views (Day, Week, Month, Next-N-Days)
- Press `Cmd+K` to convert an event into a task or vice-versa
