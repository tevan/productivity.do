# Keyboard shortcuts

Press `?` anywhere in the app to see this list inline.

## Navigation

| Key | Action |
|---|---|
| `T` | Today |
| `J` / down | Next period |
| `K` / up | Previous period |
| `G` | Go to date |
| `1`-`4` | Switch view (by tab order) |
| `D` / `X` / `W` / `M` | Day / Next-N-Days / Week / Month |

## Creating

| Key | Action |
|---|---|
| `N` or `C` | New event (uses hovered slot if any) |
| `/` | Focus quick-add input |
| `Cmd+K` | Convert event ↔ task |

## Search and find

| Key | Action |
|---|---|
| `Cmd+F` | Search events |
| `F` | Find a free time across calendars |

## Sets and toggles

| Key | Action |
|---|---|
| `Cmd+1`-`3` | Switch calendar set |
| `Esc` | Close popover/modal |
| `?` | Show all shortcuts |
