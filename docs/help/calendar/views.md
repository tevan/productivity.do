# Calendar views

Four built-in views:

- **Day** — single column, hour-by-hour. Best for focused planning
- **Next N Days** — rolling 3-7 days from today. Configurable in Settings
- **Week** — Mon-Sun or Sun-Sat (your choice). Optional weekend hide
- **Month** — full month grid with event bars

Hide views you don't use in Settings → General → Available Views. The
default view (when you first open the app) is also there.

## Switching views

- `1` `2` `3` `4` — by tab order
- `D` — Day, `X` — Next-N-Days, `W` — Week, `M` — Month
- Or click the view switcher in the toolbar

## Navigation

- `T` — jump to today
- `J` / down arrow — next
- `K` / up arrow — previous
- `G` — go to specific date

## What appears on the calendar

- Events (from connected calendars + native)
- Tasks with due times (as small chips on the day)
- All-day tasks (in the all-day row at the top)
- Travel-time blocks (between back-to-back events with locations)
- Focus blocks (your protected time)
- Subscribed ICS feeds (read-only)

Hide individual calendars from the sidebar. Hide all-day tasks via
the prefs menu.
