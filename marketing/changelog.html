Changelog

What's new. Every release we ship.

April 29, 2026

Calendly parity + public developer API

  • NewBooking pages → full Calendly parity. Multiple meeting types per page, custom invitee questions, group events, single-use invites, time polls, routing forms, branded pages, ICS download + add-to-calendar buttons, embed snippets.
  • NewPublic developer API at /api/v1. Bearer API keys with scoped permissions, OpenAPI 3.1 spec, persisted rate limiting, signed outbound webhooks with replay protection, full developer docs page.
  • NewWorkflow webhooks. Fire Slack DMs, n8n flows, or your own endpoint on booking events. Templated bodies with {{name}}, {{start}}, etc.
  • SecurityAtomic transaction-wrapped booking writes (eliminates double-booking + double-redeem races). CSPRNG for all tokens. Timing-safe API key comparison. SSRF guards on user-supplied webhook URLs. Replay-safe HMAC with timestamp.
  • ImprovedDST-safe timezone math. ICS line folding handles UTF-8 multi-byte characters. Booking page list is one query instead of N+1.
April 28, 2026

Drag-to-move, day view, group-by, booking pages v1

  • NewDrag-to-move events (vertical = retime, horizontal = day-shift). Resize via bottom edge. 15-min snap; Shift disables.
  • NewDay view as a single-column option, configurable per user.
  • NewSidebar task grouping by date / project / label / priority.
  • NewBooking pages v1 (single meeting type, basic availability, confirmation emails).
  • ImprovedInline title editing (double-click event chip), right-click context menu, multi-select tasks with bulk actions.
April 24, 2026

Light theme, settings panel, Todoist v1 migration

  • NewLight theme that follows OS preference, with manual override.
  • NewFull Settings panel with tabs: General, Calendar, Sidebar, Bookings, Weather, Time Zones, Notifications, Shortcuts.
  • FixMigrated to Todoist API v1 (project change now uses dedicated /move endpoint).
  • ImprovedCalendar visibility persists to server (cross-device sync).
April 12, 2026

Initial release

  • Google Calendar two-way sync, Todoist two-way sync, Day / Next-N / Week / Month views, calendar sets, keyboard shortcuts, basic event CRUD.
