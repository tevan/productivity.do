A day surface for plan, capture, schedule — plus a Calendly-style booking page. No email inbox, no chat, no file storage. The day surface is where those aren't.

📅 Calendar

A full Google Calendar replacement with the polish of Fantastical and the keyboard-driven speed of Vim.

Four views, one keystroke

Day, Next-N-Days, Week, Month — switch with 1, 2, 3, 4 or D, X, W, M.

Drag, drop, resize

Move events vertically to retime, horizontally to shift days. Drag the bottom edge to resize. 15-minute snap, hold Shift for free placement.

Inline editing

Double-click any event chip to rename in place. Right-click for full context menu — duplicate, change color, move calendars, delete.

Calendar sets

Group your calendars by context (Work / Personal / Projects). Switch sets with ⌘1 / ⌘2 / ⌘3. Per-set visibility and colors.

Natural language input

Type "Lunch with Sarah Tuesday 1pm" and it parses date, time, duration, and attendee. Press N from anywhere to start.

Multi-timezone bar

See your team's time zones at a glance. Half-hour offsets handled (Mumbai, Kathmandu). World clock built in.

🔗 Booking pages

Calendly's $20/mo Teams feature set, included on Pro. Public scheduling links, private invites, group events, and full automation.

Multiple meeting types

Offer 15-min discovery calls, 60-min deep dives, and 30-min syncs all from one page. Each type has its own duration, buffers, and price.

Smart availability

Per-weekday windows + date overrides + buffer times + min-notice + max-advance + daily/weekly caps. Conflicts checked across all your calendars.

Custom questions

Text, dropdown, checkbox, long-form. Required or optional. Per-page or per-meeting-type. Answers attach to the calendar event.

Group events

Run office hours or webinars with capacity > 1. Invitees see "X spots left" before they book.

Time polls

Doodle-style. Send a few proposed times to an invitee, they pick what works, you confirm — booking auto-creates.

Routing forms

Multi-page funnels. "Are you a customer or prospect?" routes to the right meeting type. Rules evaluated server-side.

Single-use invite links

Generate one-time tokens. Each link can be redeemed for exactly one booking, then expires.

Stripe at booking

Charge for sessions before the calendar event is created. Money first, meeting after.

Branded pages

Your logo, cover image, brand color. Custom domains on Team. White-label options for agencies.

Embed anywhere

Inline iframe, popup button, or direct link. The embed.js loader handles modal, focus, dismissal automatically.

Workflow webhooks

Fire a Slack DM, n8n flow, or your own endpoint when bookings happen. Templated bodies with {{name}}, {{start}} vars.

ICS & add-to-calendar

Confirmation emails ship with .ics attachments. Buttons for Google Calendar, Outlook, Apple Calendar.

✓ Tasks

Two-way Todoist sync. Tasks live alongside events — same view, same shortcuts.

Group however you think

Date, project, label, priority. Switch with one click. Each grouping shows counts and lets you bulk-act on entire groups.

Multi-select & bulk action

Shift-click to range-select, ⌘-click to toggle. Bulk: Complete, Today, Tomorrow, No date, Delete.

Full task editor

Content, description, due date+time, project, priority (with color codes), labels — all without leaving the calendar.

All-day tasks

Tasks anchored to a date show in the all-day row of your day/week views, alongside calendar events.

🤖 AI scheduling

Block time for tasks automatically. The scheduler respects your working hours, focus blocks, and existing meetings.

Auto-block tasks

Mark a task as "needs 2 hours" — the scheduler finds the next open slot that fits. Reschedules itself if conflicts appear.

Working hours

Define your real working hours. AI never schedules during evenings, lunch, or your kid's pickup window.

Focus blocks

Protect deep-work time. Booking pages won't surface those slots; AI scheduling won't fill them.

Smart rescheduling

When a meeting is added that conflicts with a scheduled task, the task moves automatically — keeping the deadline intact.

🔌 Developer API

Real, scoped, signed. Every plan gets API access — even Free.

OpenAPI 3.1

Auto-generated spec at /api/v1/openapi.json. Use any code generator. Stable contract, semver guarantees.

Scoped API keys

Read or write per resource. read:bookings, write:tasks, etc. Revoke or rotate from settings.

Signed webhooks

HMAC-SHA256 with timestamps to prevent replay. Retry queue with exponential backoff. Receivers can rotate secrets without downtime.

Rate limits

60/min Free, 120/min Pro, 600/min Team. Persisted across server restarts. Standard X-RateLimit-* headers.

Embed snippets

Drop a <script src="/embed.js"> tag on any site. Auto-wires data-productivity-book elements. Popup or inline.

Self-hostable

Open-source backend. Run on your own infrastructure. License included on Team plan.
